# linux-dotfiles

## Getting started

```bash
git clone https://github.com/casonadams/linux-dotfiles.git
cd linux-dotfiles
git submodule add https://github.com/deadc0de6/dotdrop.git
python3 -m pip install -r dotdrop/requirements.txt --user
```

## Updating

```bash
dotdrop update
```

## Adding files

```bash
dotdrop add <file_location>
```
