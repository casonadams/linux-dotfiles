# linux-dotfiles

## Getting started

```bash
git clone https://github.com/casonadams/linux-dotfiles.git
cd linux-dotfiles
git submodule update --init
python3 -m pip install -r dotdrop/requirements.txt --user
```

## Updating

```bash
dotdrop update
```

## Adding files

```bash
dotdrop import <file_location>
```
